## ✨ About the Project

This portfolio is built as a lightweight and fast-loading website. It is automatically updated to reflect the latest changes and deployment status.

## 🛠️ Built With

*   **HTML5**: For the core structure and content of the site.
*   **GitHub Actions**: For automating the timestamp update process.

## 🚀 Features

*   **Clean, Simple Design**: Focuses on presenting content clearly.
*   **Automated Timestamp**: The `updates.txt` file is automatically updated via a GitHub Actions workflow to show the latest deployment time (e.g., `Fri Mar 6 18:26:54 UTC 2026`).

## 📁 Repository Structure

*   `index.html` - The main entry point and content for the portfolio website.
*   `updates.txt` - A text file containing the latest update timestamp, managed by automation.
